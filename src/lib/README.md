# Lib - репозиторий модулей.

## check_formats:

Репозиторий с модулями для проверки формата ввода данных.

## classes:

Репозиторий с готовыми моделями классов.

## json_csv_xlsx:

Репозиторий с модулями для работы с JSON, CSV и  XLSX.

## Module for List:

Репозиторий с модулями для обработки и анализа списков.

## Module for Text:

Репозиторий с модулями для обработки текста, строк.

## Отдельные модули:

### isInteger.

Модуль для проверки, является ли переменная числом.

### pathProccesing.

Модуль обработки path. Существует ли он или нет,  и есть ли необходимость его создавать или нет.
А также имеет функции проверки расширения. Типы расширенний указывается в path_suffix.

p.s. inicialPath - упрощённая версия pathProccesing

### stack_queue.

Модуль для работы с очередями и стеками.

### linklist.

Модуль для работы с односвязными списками и узлами.
