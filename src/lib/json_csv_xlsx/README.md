# json_csv_xlsx:

## json_csv_xlsx

### Модуль с следующими функциями:

#### csv_to_json

Конвертирует файл формата CSV в файл формата JSON.

#### csv_to_xlsx

Конвертирует файл формата CSV в файл формата XLSX.

#### json_to_csv

Конвертирует файл формата JSON в в файл формата CSV.

## json_csv_xlsx_auto

Аналогично предыдущему модулю, но работает автоматически, то есть одна функция для всех случаев (json <-> csv и csv -> xlsx).
