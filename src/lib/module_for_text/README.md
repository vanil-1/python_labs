# Module for Text:

## normalize

Обрабатывает строку и "нормализирует её" (все заглавные буквы -> строчные, все ё/Ё -> е/Е и т.д.).

## tokenize

Разбивает строку на токены.

## count_freq

Подсчитывает количество каждого токена и сортирует их в порядке убывания частоты, а также, в случае, если частота разных токенов совпадает, то сортирует в алфавитном порядке.

## top_n

Продолжение модуля count_freq, но вместо всего словаря выдаёт превые 5 токенов по частоте.

## text_token

Модуль, объединяющий все 4 предыдущих модуля в одном.

## text_analizator

Модуль выполняет те же функции, что и модуль text_token, но с небольшим визуальным оформелнием, то есть выходные данные могут быть в виде текста или в виде таблицы.
